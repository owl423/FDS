### Ajax using Vue JS

`vue-resource` 모듈 설치

```sh
$ npm i -D vue-resource
```

`vue-resource` 모듈 로드 및 사용 설정

```js
import VueResource from 'vue-resource';
Vue.use(VueResource);
```

```js
this.$http.post('https://vue-js-ajax.firebaseio.com/data.json', {data})
          .then(response=>{
            // response
          }, error=> {

          });
```
